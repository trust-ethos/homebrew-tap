# Ethos Homebrew Tap

Official Homebrew tap for [Ethos CLI](https://github.com/trust-ethos/ethos-cli).

## Installation

```bash
brew tap trust-ethos/tap
brew install ethos
```

## Updating

```bash
brew update
brew upgrade ethos
```

## Available Formulae

- `ethos` - CLI for querying Ethos user profiles, XP balances, and reputation data
